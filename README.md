# React + TailwindCSS + Vite.js + AWS 

This repo will scaffold a new React-project using Vitejs to get React and TailwindCSS up-and-running.
And host it in AWS.

Project uses `Projen`.

```shell
yarn install
```

```shell
cd backend && cdk deploy
```
